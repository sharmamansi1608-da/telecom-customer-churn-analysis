# Telecom Customer Churn Analysis

##  Objective
Analyze customer churn behavior and identify key drivers impacting revenue loss. The project focuses on segment-level churn patterns and provides data-driven retention strategies.

##  Key Insights
- Churn rate is ~26.5%, impacting ~$3.68M revenue
- High-value customers contribute the highest revenue loss (~$2.09M)
- Month-to-month contracts show highest churn (~46%)
- Early-stage churn (0–6 months) is highest (~53%)
- Competitor-driven churn accounts for ~46%

##  Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI

##  Project Structure
- telecom_churn_analysis.ipynb → Full analysis
- telecom_customer_churn.csv → Dataset
- telecom_dashboard.pbix → Dashboard

##  Key Features
- KPI analysis (churn rate, revenue at risk)
- Customer segmentation (ARPU-based)
- Contract & tenure analysis
- Root cause analysis (churn reasons)
- Business recommendations
